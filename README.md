# How To Use this Dark Mode (Before Published to Chrome Extension)
- Download both of the codes
- Go to chrome://extensions/
- Or go to extension manager on top right near your profile
<img src="https://pbs.twimg.com/media/GnlUPpcboAAq4Op?format=jpg&name=4096x4096" width="1100">
- Enable the developer mode on top right side
<img src="https://pbs.twimg.com/media/GnlWF71bEAA4kaF?format=jpg&name=4096x4096" width="1100">
- Click on the "Load Unpacked" button on top left side
<img src="https://pbs.twimg.com/media/GnlWhH4aYAAUv-y?format=jpg&name=4096x4096" width="1100">
- Select the folder where you save both of the codes
- If it succesfully installed, the appearance should be like this image. (Could be shown more if you had more extension)
<img src="https://pbs.twimg.com/media/GnlXONpaMAAsXAt?format=png&name=large" width="1100">
- Check your LMS website and find the button to toggle on/off the dark mode on top right side
<img src="https://pbs.twimg.com/media/GnlXke-aMAIZYxP?format=jpg&name=large" width="1100">
